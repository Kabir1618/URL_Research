Welcome to the Research Tool repository! This project is an intelligent platform designed for efficient research and question-answering, leveraging the power of OpenAI and Pinecone technologies. It enables users to input article URLs, process them, and generate insights using advanced language models.

## Overview

The Research Tool is an interactive application that simplifies the research process by offering dynamic URL processing, intelligent text splitting, and efficient question-answering capabilities. Whether you're a student, researcher, or knowledge enthusiast, this tool streamlines your information retrieval and analysis.

**Access the Chatbot:**
   - Visit the [Hugging Face Spaces](https://huggingface.co/spaces/Kabirsingla/URL_LLM) deployment.
   - Start a conversation by typing your questions and interact with the chatbot.
   - 
## Features

- **Dynamic URL Processing:** Easily input article URLs for seamless data extraction.
- **Intelligent Text Splitting:** Utilizes RecursiveCharacterTextSplitter for optimal document segmentation.
- **Embedding and Indexing:** Leverages OpenAIEmbeddings to create embeddings and Pinecone to build an embedding vector index.
- **Question-Answering Chain:** Employs OpenAI's language model for effective question-answering capabilities.

## How to Use

1. Provide URLs in the sidebar.
2. Input questions to retrieve insightful answers using the integrated language model.
3. Press Enter to initiate data loading, text splitting, embedding vector index building, and getting your answer .


## Technologies

- Streamlit for the user interface.
- OpenAI for advanced language processing.
- Pinecone for efficient vector indexing.

## Dependencies

- Python 3.x
- Install dependencies with `pip install -r requirements.txt`.
- Set up environment variables for OpenAI and Pinecone API keys in a `.env` file.
